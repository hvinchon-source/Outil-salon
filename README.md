# 🎯 ExpoLeads Pro

Application Streamlit pour extraire et qualifier les exposants de salons professionnels.

## Fonctionnalités
- Scraping multi-mode (Auto, Infinite Scroll, Pagination)
- Qualification ICP automatique (Auto, DIY, Pharma, HAP, Petfood, Jouet)
- Séparation Marques / Distributeurs
- Détection e-commerce
- Génération automatique de liens LinkedIn
- Export Excel
